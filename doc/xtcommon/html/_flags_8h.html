Register a new enum-based flags macro.

Should be used to get access to AND/OR/XOR logical operators with a given enum values.

Example:

enum class MyFlags
{
None = 0x0,
One = 0x1,
Two = 0x2,
Three = 0x4,
Four = 0x8
};
ENUM_FLAGS(MyFlags)
int main()
{
auto mask = MyFlags::One | MyFlags::Two;
